# 함평군 · 전남광주통합특별시

> 함평군 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-jeonnamgwangju-hampyeong.github.io/**

함평군(咸平郡)은 전남광주통합특별시 서북부, 서해 함평만에 면한 1읍 8면의 농·어촌 군으로 옛 전라남도에 속했던 고장이다. 봄이면 함평엑스포공원 일대에서 17종 20만 마리의 나비와 50만 본의 봄꽃이 어우러지는 함평나비대축제가 열려 '나비의 고장'으로 널리 알려졌고, 가을에는 천만 송이 국화가 만발하는 대한민국 국향대전이 이어진다. 청정한 갯벌과 게르마늄 토양에서 무항생제 조사료로 키운 '함평천지한우'는 2008년 전국 최초로 한우산업특구로 지정될 만큼 명성이 높다. 돌머리해수욕장의 서해 낙조, 남한에 유일하게 남은 고려시대 다리 고막천 석교, 1999년 재발견된 천연기념물 붉은박쥐(황금박쥐)까지, 함평은 작지만 자연과 축제·먹거리가 촘촘히 어우러진 독특한 고장이다. 인구는 약 2만 9천여 명으로 소멸을 걱정하는 전형적 농촌이지만, 축제 기간이면 수십만 관광객이 몰리는 브랜드 관광 도시로 거듭났다.
